# 変更履歴

## [0.4.0] - 2026-04-26

### 追加
- ハイブリッド画像保存方式のサポート（オリジナル画像を別ファイル `.img_*.png` として保存）
- サムネイル画像の最適化機能（プレビュー用の Base64 画像サイズを自動縮小）
- Kakidash クラスに新規 API を追加: `getImages()`, `gcImages()`, `zoomNode()`

### 変更
- kakidash ライブラリを v0.4.0 に更新

### 修正
- ファイル保存および同期時の画像データ破損を防止

## [0.0.11] - 2026-03-17

### 追加
- 取り消し線のショートカット `s` を追加

### 変更
- kakidash を 0.3.2 に更新
  - 取り消し線（strikethrough）テキストスタイルのサポート
  - タッチデバイスサポートのためのポインターイベントへの移行
  - 意図しないドラッグ動作を防止するための距離ベースのしきい値を導入
  - クリップボードの貼り付け不具合および Markdown エクスポートダイアログの修正
  - 大規模なアーキテクチャのリファクタリング（InteractionOrchestrator, CommandBus 等の実装）
